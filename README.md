# Hotel-Booking-Cancellation-Analysis (Excel)
Analyzed Hotel Booking Cancellations using Excel and Data Analytics

## Project Overview
This project analyzes hotel booking data to understand cancellation patterns and key factors influencing booking cancellations.  
The analysis and dashboard were created using Microsoft Excel.

Dataset source: WSCube Tech (YouTube)

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Slicers
- Data Cleaning Techniques

---

## Dataset Description
The dataset contains hotel booking records with details such as:
- Hotel Type (City Hotel / Resort Hotel)
- Booking Status (Canceled / Not Canceled)
- Lead Time
- Arrival Month & Year
- Market Segment
- Customer Type
- Deposit Type
- Country

---

## Dashboard Highlights
The Excel dashboard includes:
- Total Bookings
- Total Cancellations
- Cancellation Percentage
- Hotel-wise cancellation comparison
- Monthly booking & cancellation trend
- Market segment wise cancellation analysis
- Customer type & deposit type impact on cancellations

---

## Key Business Insights
- City Hotels have a higher cancellation rate compared to Resort Hotels.
- Longer lead time bookings are more likely to be canceled.
- Bookings without deposit show higher cancellation probability.
- Online Travel Agents (OTA) contribute the most to cancellations.
- Certain months show peak cancellation trends indicating seasonality.
